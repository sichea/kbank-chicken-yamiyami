필수 파일 다운로드

아래의 버튼을 클릭하여 제가 직접 세팅하고 검증한 핵심 파일들을 즉시 다운로드하세요!

AutoX.js 우회용 APK 설치 파일 Autox.apk (Knox 차단 해제 검증됨)
진짜 닭 이미지 (타겟용) chicken.jpg (정밀 크롭 완료)
  1. 다운로드 받은 파일 관리

    APK 파일은 PC에서 다운받아 WebADB 설치에 사용하고, 닭 이미지(chicken.jpg)는 스마트폰의 내장 메모리 > Download 폴더로 복사해 줍니다.

닭 이미지의 최종 스마트폰 경로는 /sdcard/Download/chicken.jpg 가 되어야 코드가 정확히 인식합니다.

스마트폰 개발자 설정 켜기

  1. 개발자 옵션 활성화

    스마트폰의 설정 ➡️ 휴대전화 정보 ➡️ 소프트웨어 정보 메뉴에서 [빌드 번호] 항목을 연속 7번 터치합니다.

  2. USB 디버깅 켜기

    다시 설정 메인 최하단에 생성된 개발자 옵션으로 들어가 [USB 디버깅] 항목 스위치를 켭니다.

컴퓨터와 폰을 케이블로 연결할 때 폰 화면에 뜨는 "USB 디버깅을 항상 허용" 창에서 반드시 허용을 눌러주셔야 합니다.

WebADB를 이용한 백신 우회 인젝션

  1. 웹사이트 연결

    폰과 PC를 USB 케이블로 연결한 뒤, PC 브라우저로 WebADB Install 웹사이트에 접속합니다.

  2. APK 강제 주입

    [Add Device]를 눌러 폰을 선택하여 연결한 뒤, Choose File 버튼으로 방금 다운받은 Autox.apk 파일을 선택하고 [Install]을 누르면 폰 백신의 차단을 뚫고 강제 설치가 완료됩니다!

최초 실행 및 권한 세팅

  1. 3대 권한 켜기

    AutoX.js 실행 후 안내에 따라 접근성 권한, 다른 앱 위에 표시 권한(플로팅 창), 화면 캡처 권한을 모두 켭니다.

  2. 스크립트 작성 및 시작

    우측 하단 [+] ➡️ [파일]을 눌러 치킨 스크립트를 생성하고, 편집 아이콘을 눌러 아래의 코드를 붙여넣고 저장합니다. 케이뱅크 게임 화면에서 플로팅 창의 [▶ 재생] 버튼을 누르면 시작됩니다!

비상 정지(킬스위치): 매크로 실행 중 긴급하게 멈추고 싶을 때는 스마트폰 옆면의 **볼륨 버튼(위 또는 아래)**을 누르면 광속으로 매크로가 즉시 강제 중지됩니다!

AutoX.js 10단계 크래셔 소스 코드

// ==========================================
// 10단계 격파용 '초광속 크래셔' 스크립트 (AutoX.js)
// ==========================================

if (!requestScreenCapture()) {
    toastLog("화면 캡처 권한 승인 실패");
    exit();
}

// ================= [설정 구역] =================
// 💡 다운로드 폴더의 닭 이미지 파일 경로
var targetImgPath = "/sdcard/Download/chicken.jpg"; 

// 💡 10단계용 극단적 임계값 설정 (70% ~ 75% 추천)
var matchThreshold = 0.73; 

// 💡 클릭 후 대기 시간 극단적 최소화 (50ms)
var delayAfterClick = 50; 

// 💡 탐색할 스마트폰 화면 중심부 영역 설정
// [X좌표 시작, Y좌표 시작, 가로폭, 세로폭] (1080x2340 폰 기준 최적화)
var searchRegion = [150, 850, 780, 600]; 
// ===============================================

var template = images.read(targetImgPath);

if (!template) {
    toastLog("이미지 파일을 찾을 수 없습니다!\n경로를 확인해주세요: " + targetImgPath);
    exit();
}

toastLog("🔥 10단계 크래셔 모드 가동! 3초 뒤 시작...");
sleep(3000);
toastLog("🚀 감지 시작! 10단계를 박살냅니다.");

while (true) {
    var img = captureScreen();
    if (!img) continue;

    // 설정된 중앙 영역에서만 이미지 템플릿 매칭 초고속 수행
    var p = images.findImage(img, template, {
        region: searchRegion,
        threshold: matchThreshold 
    });

    if (p) {
        var clickX = p.x + (template.getWidth() / 2);
        var clickY = p.y + (template.getHeight() / 2);
        
        // 일반 탭 대신 5ms 광속 클릭 터치 이벤트 주입
        press(clickX, clickY, 5); 
        
        sleep(delayAfterClick); 
    }
}
